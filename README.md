# 💙나의 장보기 스타일(나장스)💙
<br />

## ✅ 프로젝트 소개
  MBTI의 성향을 기반으로 장보기 스타일을 구현했습니다.
<br /><br />

## 🧑‍🤝‍🧑 멤버 구성
  - 개발/디자인 1명
  - 기획자 1명
  - 데이터 분석가 1명
<br /><br />

## 🔗 배포 링크

<br /><br />

## ⚙ 개발 환경
  - `React`
  - `JavaScript`
  - `CRA`
  - `Styled-Components`
<br /><br />

## 📌 주요 기능

### 1. 메인 페이지
  - 테스트 시작
  - 테스트 공유
    
### 2. 질문 페이지
  - 12개의 질문
  - 두 가지의 보기 중 택1
    
### 3. 결과 페이지
  - 나의 장BIT
  - 결과에 따른 상세 풀이
  - 16가지의 결과 도출
  - 카카오톡으로 내 결과 공유하기
  - 클립보드
  - 테스트 재시작

<br /><br />

## 👩🏻‍💻 특징 및 부가 설명
  - PoC 프로젝트 중 하나, 회사에서 서비스 중인 앱의 유입 및 홍보를 목적으로 제작
  - MBTI 기반으로 사용자 장보기 스타일을 분석하고 쇼핑할 때의 선호도와 습관을 고려한 결과 도출
  - 테스트 카카오톡 공유 / 테스트 결과 카카오톡 공유
  - 총 테스트 사용자 수 : 1,434명
  - 앱 유입 수 : 63명
